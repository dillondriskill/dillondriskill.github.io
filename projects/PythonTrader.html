PythonTrader

Source

This is one of my oldest serious coding projects, and I see it as my first serious coding project, I started work on it when during my junior year of highschool and finished it during my senior year, when I used it for my capstone project. I coded it entirely in Python and learned a lot about software development, automated stock trading (aka algorithmic trading), and data science. I spent a lot of time on this project, and hope to one day make a better version, perhaps using C or C++ for better speed and a little extra challenge, while at the same time incorporating better coding practices and knowledge that I have gained (and will continue to gain) since I finished this project.
