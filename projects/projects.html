PicoRocket

Source Project page

This has been a long time project of mine where I am aiming to build a self guided model rocket from scratch using a Raspberry Pi Pico as the main guidance computer. In this project I explore concepts of rocket design (duh), multicore embedded processing, low level I/O, terminal interfaces (the raw kind), and much, much more.
